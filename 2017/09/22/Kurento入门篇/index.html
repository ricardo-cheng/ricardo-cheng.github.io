Kurento 简介

Kurento 是一个 WebRTC 媒体服务器和一套客户端 API

    Kurento 简化了对 WWW 和智能手机平台的高级视频应用的开发。Kurento 的功能包括视听流的群组通信，代码转换，录​​音，混音，广播和路由。

    Kurento 还提供涉及计算机视觉，视频索引，增强现实和语音分析的高级媒体处理能力。Kurento 模块化架构简化了第三方媒体处理算法（即语音识别，情感分析，人脸识别等）的集成，应用程序开发人员可以将其视为 Kurento 内置功能的其余部分。

KMS

KMS 是 Kurento Media Server 的缩写，它是 Kurento 的核心要素，负责媒体传输，处理，加载和录制

KMS 基于 GStreamer 的低级技术实现，以优化资源消耗。它提供以下功能：

  1. 网络流式传输协议，包括 HTTP，RTP 和 WebRTC
  2. 支持媒体混合和媒体路由/调度的组通信（ MCU 和 SFU 功能）
  3. 对计算机视觉和增强现实滤波器的通用支持
  4. 媒体存储支持 WebM 和 MP4 的写入操作，并 以 GStreamer 支持的所有格式播放
  5. 由 GStreamer 支持的任何编解码器之间的自动媒体转码，包括 VP8，H.264，H.263，AMR，OPUS，Speex，G.711 等

Media Element

Media Element 是封装特定的媒体功能的模块

Media Pipeline

Media Pipeline 是一组 Media Element，其中由一个元素（源）产生的输出流被注入到一个或多个其他元素输入流中
